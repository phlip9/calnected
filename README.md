Calnected
=========


Resources
---------

http://www.ocf.berkeley.edu/~vlo/BerkeleyResources.php


Setup
-----

Clone the repo

    $ git clone git@github.com:phlip9/calnected.git

Install Flask

    $ pip install Flask

Run the app

    $ python3 app.py

Hit up the webpage

    $ google-chrome 127.0.0.1:5000/
